# iphone
![iphone](https://user-images.githubusercontent.com/35535783/65541901-24052880-dedc-11e9-8cbe-0d98f063c005.PNG)

iPhone UI on the web.

Built with:
 * Javascript, HTML/CSS

To run locally:

```$ git clone https://github.com/rileythomp/iphone.git```

```$ cd iphone```

```$ npm i```

```$ npm start```

Then go to localhost:3000.
